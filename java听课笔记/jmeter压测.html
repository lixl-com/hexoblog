测试术语

RT: Response Time 响应时间，指客户端从发起请求到收到服务器响应结果的时间。

并发数：指服务器同时能处理的请求数量。

QPS ：QueriesPerSecond 每秒查询率，即一台服务器每秒处理的请求数量。

TPS：TransactionsPerSecond每秒事务率，一个事务对应一个或多个请求(如一个web页面上可能有多个请求)。

吞吐量：是一个服务器处理请求速度的综合指标(服务器的cpu、内存、硬盘、网卡等组件都会影响到这个指标)，

如网关(用来转发请求)-主要涉及cpu、内存、网卡；

数据库(oracle/mysql)-cpu、内存、硬盘、网卡等
